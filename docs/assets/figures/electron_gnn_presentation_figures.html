Projector:

Export and publication workflow (from web + tooling research)

  • SVG strengths: infinite zoom, editable text, journal-friendly when embedded in PDF or supplied as separate vector (see e.g. svgutils for composing panels in Python).
  • This file: Use browser Print → Save as PDF for a quick handout. For LaTeX, export individual panels: open DevTools, select an svg element, copy outerHTML, save as .svg, then \\includesvg (svg package) or convert to PDF in Inkscape.
  • Mermaid → SVG: Keep Mermaid sources in docs/ONE_SHOT_PREDICTION_QA_DIAGRAMS.md; paste into mermaid.live and export SVG/PNG for slides.
  • Plots (spectra, loss): Prefer matplotlib SVG output (plt.savefig("fig.svg")) for data curves; compose with Inkscape or svgutils.
  • MCP note: There is no universal “diagram MCP” in this repo. Optional Cursor MCPs (e.g. design tools) are account-specific; for reproducible papers, version-controlled SVG/HTML plus scripted matplotlib remains the most portable path.
